## Requirement
*required

- node* version 22.14.0
- npm* verision 10.9.2
- nest-cli* ```npm install -g @nestjs/cli```
- vite* ```npm install -g vite```
- postgres*
- docker (alternate)

## Project setup

```bash
$ npm install
$ cd frontend
$ npm install
```

## Compile and run the project

```bash
# start project
$ cd frontend
$ npm run build
$ cd ..
$ npm run build
$ npm run start
```


## Deployment

To deploy this app just simply run

```bash
# docker for production mode
$ docker compose up -d
```
